Projects

Lambda Labs - Get2it

This project was a collaborative effort of 5 Lambda students. Using GitHub to merge our code, the project is built using React, Redux and Axios on the front end while the back end uses Node.js, Express, Knex.js and PostgreSQL. Sign up, login, create/read/update/delete tasks and have in app notifications remind you when a task is about to start.

Back-End Build Week - One Line a Day

Build Weeks are 2 week long sprints at the end of a unit meant to let students build a project from scratch using all the skills they learned. For the Back-End build week I created a Back-end server with RESTful API endpoints and a relational database for persistent data and authentication. Built using Node.js, Express, Knex.js, bcrypt.js, nodemon, sqlite3.

UI/UX Build Week - FIFA Player Stats

Build Weeks are 2 week long sprints at the end of a unit meant to let students build a project from scratch using all the skills they learned. For the UI/UX build week we created a responsively designed marketing page for an app to view FIFA player stats. Built using LESS preprocessor, media queries and DOM manipulation with JS query selectors. Click the player names to expand.
